# Vite + React Architecture

> This skill is under development.

Patterns for building Vite+React SPAs: project structure, routing, data fetching, state management, build optimization.

## Topics to Cover

- Project structure conventions (`src/`, `pages/`, `components/`, `hooks/`, `lib/`)
- React Router v6+ with lazy-loaded routes
- Data fetching patterns (TanStack Query integration)
- State management tiers (local, shared, server)
- Vite configuration: path aliases, env variables, proxy, build optimization
- Code splitting and bundle analysis
- Environment-specific configuration
