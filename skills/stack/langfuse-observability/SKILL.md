# Langfuse Observability

> This skill is under development.

Patterns for instrumenting LLM applications with Langfuse for tracing, cost tracking, and evaluation.

## Topics to Cover

- Trace decoration with `@observe()` decorator
- Span nesting for multi-step pipelines
- Cost tracking per trace and per user
- Prompt management and versioning via Langfuse
- Evaluation pipelines (scoring, human feedback)
- Integration with LangChain/LangGraph callbacks
- Dashboard organization and filtering
- Flushing traces before process exit
